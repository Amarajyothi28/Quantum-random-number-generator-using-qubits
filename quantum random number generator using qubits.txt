This project creates a Quantum Random Number Generator (QRNG) using Qiskit, a free Python tool for quantum computing. It uses one qubit—a quantum bit that can be both 0 and 1 at once. We start the qubit at 0, then add a Hadamard gate. This splits it into equal chances: 50% for 0 and 50% for 1, thanks to quantum superposition. When measured, it randomly picks one, giving true randomness. Unlike regular computer random numbers, which use math formulas and repeat patterns, this comes from quantum physics and can't be predicted.

We run a simulation 10,000 times on a virtual quantum computer (no real hardware needed). A histogram shows nearly even 0s and 1s, proving it works. The full code is simple—just 20 lines—to copy, install Qiskit, and run yourself. Extensions include more qubits for bytes, IBM Quantum cloud access, or NIST tests for pro-level randomness checks. Perfect for secure passwords, games, simulations, or quantum learning. (148 words)
